### AI R&D — аналитика сессий PAM 

- Проанализировать возможные сценарии применения AI для обработки данных сессий PAM
- Оценить возможность использования OCR для извлечения текста из видеозаписей RDP
- Оценить возможность анализа команд и вывода в SSH-сессиях с использованием NLP
- Исследовать возможности нейросетей из списка заказчика
- Исследовать методы выявления отклонений в поведении пользователей
- Проработать архитектурный вариант подключения AI-аналитики к PAM.
- Сформировать итоговый отчёт по AI R&D (Отчёт должен содержать: список возможных AI-сценариев; ограничения и риски; ориентировочную оценку сроков и трудозатрат; рекомендации: делать / не делать / отложить.)

### LLM которые предлагается использовать

- Gemma – мультимодальная нейросеть (анализ текста и изображений).
- T-Pro 2.1 – анализ текстов.
- GPT OSS от OpenAI – открытая модель от OpenAI(анализ текста).
- Qwen Coder – специализированная модель обученная на работу с кодовой базой ПО.
- Звук: Whisper – преобразование аудиозаписей в текст.
- Видео: Yolo 11 – анализ видеопотоков (для специализированных задач необходимо дообучение модели).
